复制原理
bin-log 二进制日志
所有DB事件先写入bin-log
Slave从Master读取bin-log

用途

  • 扩展：Slave分担master负载
  • 分析：数据分析不影响Master
  • 备份：作为一种数据备份手段
  • 分布：异地保存数据副本（也可以实现应用访问本地库）
